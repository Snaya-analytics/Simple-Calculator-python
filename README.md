# Simple-Calculator-python
This is a simple calculator program built using python.
It perfroms basic arithmatic operations like addtion, subtraction, multiplication and division.

## Features
- Perform addtition of two numbers
- Perform subtraction of two numbers
- Perform multiplication of two numbers
- Perform division of two numbers
- Simple and beginner-friendly python code 

## How to Run
1. Install python on your computer.
2. Download or clone this project from github.
3. Open the project folder in your terminal or VS code.
4. Run the program using:
5. simple calculator.py
